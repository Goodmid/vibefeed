# VibeFeed

## 🇬🇧 English

A simple social feed application inspired by Twitter, built with React.
Users can create posts, like them, add to favorites, and search through the feed.

📖 About

VibeFeed is a React-based application that mimics a simplified social media feed.
It allows users to interact with posts in real-time by liking, favoriting, adding, and searching content.

The project focuses on component-based architecture, state management, and user interaction.

🚀 Features:

📝 Add new posts
❤️ Like posts
⭐ Add posts to favorites
🔍 Search posts
🗂 Filter posts (liked / important)
⚡ Instant UI updates
📱 Responsive interface

🛠 Tech Stack:
React
JavaScript (ES6+)
CSS / SASS
Webpack
Babel

📦 Installation & Setup

Clone the repository:

git clone https://github.com/YOUR_USERNAME/vibefeed.git

Go to the project folder:

cd vibefeed

Install dependencies:

npm install

Run the app:

npm start

Open in browser: http://localhost:3000

## 🇷🇺 Русский

Простое приложение для работы с лентами социальных сетей, вдохновленное Twitter и созданное на React.
Пользователи могут создавать публикации, ставить лайки, добавлять в избранное и искать контент в ленте.

📖 О проекте

VibeFeed — это приложение на основе React, имитирующее упрощенную ленту социальных сетей.
Оно позволяет пользователям взаимодействовать с публикациями в режиме реального времени, ставя лайки, добавляя в избранное, добавляя контент и осуществляя поиск.

Проект ориентирован на компонентно-ориентированную архитектуру, управление состоянием и взаимодействие с пользователем.

🚀 Особенности:

📝 Добавление новых постов
❤️ Лайки

⚡ Мгновенные обновления пользовательского интерфейса
